### Create package.json file:
```
npm init -y
```

### Install Jest:
```
npm i -D jest
```

### Run tests:
```
npm test
```
Test suites are all the test files.

### Install Axios:
```
npm i axios
```

### To turn on watch mode:
```
npm run test -- --watch
```
